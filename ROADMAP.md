# zack-shell 开发路线图

一个类 FinalShell 的 SSH/SFTP 终端 + 服务器监控工具。

**技术栈**：Electron + React + TypeScript（electron-vite）/ ssh2 / xterm.js

共 **7 个阶段**，每个阶段都能独立跑通验证。

---

## ✅ 阶段一：地基（已完成）

工程脚手架，三进程分层（main / preload / renderer）。

- [x] Electron + React + TS + Vite 工程
- [x] 主进程 / preload 安全桥 / 渲染进程结构
- **验证**：`npm run dev` 弹出窗口

## ✅ 阶段二：SSH 终端 + 会话管理（已完成）

- [x] ssh2 建连、开 shell 通道、双向数据转发
- [x] xterm.js 终端，尺寸自适应
- [x] 多标签（同时多台 / 同一台多开），后台标签输出不丢失
- [x] 主机列表侧边栏：分组、增删改、连接
- [x] 本地持久化，密码/口令用系统密钥库（safeStorage）加密
- [x] 密码 + 私钥两种登录，keyboard-interactive 回退
- **验证**：重启后主机还在，点击即连

## ✅ 阶段三：服务器监控面板（已完成）

复刻截图左侧那一栏，定时跑远程命令采集系统指标。

- [x] 采集后端：每 2s exec `/proc/stat`、`/proc/meminfo`、`/proc/net/dev`、`ps` 等并解析
- [x] CPU / 内存 / 交换分区 实时百分比 + CPU 折线图
- [x] 网络上下行速率
- [x] 进程 TOP 列表（按内存）
- [x] 磁盘挂载点占用
- [x] 系统信息（IP、运行时长、负载）
- **验证**：数字随服务器负载实时变化

## ⬜ 阶段四：SFTP 文件管理

- [ ] SFTP 会话（复用同一 SSH 连接）
- [ ] 远程目录树浏览，本地双栏
- [ ] 上传 / 下载（带进度）
- [ ] 新建 / 重命名 / 删除 / 改权限
- [ ] 拖拽传输
- **验证**：传一个文件上去并在终端 `ls` 看到

## ⬜ 阶段五：编辑与便捷功能

- [ ] 远程文件双击在线编辑（下载→编辑→回传）
- [ ] 终端搜索、复制粘贴增强
- [ ] 命令历史 / 快捷命令
- [ ] 标签拖拽排序、断开重连

## ⬜ 阶段六：体验优化

- [ ] 主题 / 配色、字体大小设置
- [ ] 多语言（中 / 英）

- [ ] 密钥/密码安全加固审查
- [ ] 设置面板

## ⬜ 阶段七：打包发布

- [ ] 应用图标、版本信息
